================================================================================
                     FERRITE RUNTIME вҖ” SYSTEM ARCHITECTURE
================================================================================

  "VRAM is not a container. It is a pipe."

  Ferrite is a GPU operating system that virtualizes video memory through
  O(1) TLSF allocation, shard-streamed execution, and wave-scheduled CUDA
  streams вҖ” enabling computations that exceed physical VRAM by 10-1000x.
